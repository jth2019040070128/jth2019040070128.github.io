安装GIT

找程序

访问 https://git-scm.com/

我的电脑win11，所以选择

下载好后，点击安装即可。

测试

打开git，输入命令 git –version

查看版本号，Git安装成功

安装Node.js

找程序

访问 https://nodejs.org/zh-cn/

根据电脑选程序，我的Win11，选择x64.mis，具体可看 https://www.cnblogs.com/zhouyu2017/p/6485265.html 这个网址

下载完成后，点击安装，我在D盘创建了node文件夹，放应用程序

安装过程中出现的选择框，我根据 https://www.cnblogs.com/zhouyu2017/p/6485265.html 博客弄得，没有就默认，我是这样

测试

安装完成后，运行Git程序，就是测试目录下的，输入命令 node -v npm -v （逐个输入） 可看到版本号

也可 在windows命令框下输入 cmd 打开命令框，输入命令 node -v npm -v （逐个输入） 可看到版本号

安装hexo

下载

在 windows命令框下 或 Git命令框下 输入 npm install express -g 即可

错误

如果出现错误可把错误粘贴，去网上搜搜，比如我就遇到了这个问题，很像这个只是不是c盘是d盘

解决也很简单 https://blog.csdn.net/qq_42780155/article/details/119025638 看这个

流成

具体流程可看 https://www.bilibili.com/read/cv5107169

创建仓库

第二步

最后点这个

出现这个既成功

关联账号

Git命令框

输入 ssh-keygen -t rsa -C “邮箱地址” 邮箱地址填自己注册时的QQ邮箱

按4次回车键 关联成功

找到.ssh文件 一般在这个文件夹下，windows11

用notepad++ 软件打开.pub文件 软件下载在这里 https://blog.csdn.net/qq6759/article/details/88384608

复制里面代码到这里

关联账号成功

Git命令行输入 ssh -T git@github.com

成功关联

本地编写文章上传

新建文件夹放博客内容

新建文件夹，右键选择Git命令框

输入 hexo init 初始化

输入 hexo s 复制网址 打开即可

成功打开 按 Ctrl+C 关闭页面

上传服务器

打开文件，更改配置

打开文件夹，，用 notepad++ 打开.yml文件

更改配置 在最后两行输入

deploy:

type: git

repository: https://github.com/jth2019040070128/jth2019040070128.github.io.git

branch: main

保存

网址在这里

命令行配置 输入 npm install hexo-deployer-git –save

成功实现代码上传
